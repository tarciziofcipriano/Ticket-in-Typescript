# Ticket in Typescript

Alguns exemplos de Ingressos feitos em Typescript (no tamanho 80mm e A4)

### Dependencias

Instale todas as dependencias usando o comando yarn

```yarn```

### Storybook Server

Rode servidor storybook.

```yarn storybook```

### Test

Rode os tests.

```yarn test```

## Build

Build projeto.

```yarn build```
